# Student Performance Modeling & Simulation

## Project Description
This project utilizes a dataset of student metrics to model academic performance through statistical visualization. Specifically, it generates a histogram of the `final_score` column to analyze the distribution of grades, which is a fundamental step in modeling and simulation to identify if the data follows a normal distribution (bell curve).

## Files in this Repository
*   `main.py`: The Python script used to process the data and generate the histogram.
*   `student_performance.csv`: The dataset containing student IDs, study hours, and scores.
*   `histogram_output.png`: The exported visualization showing the frequency of student scores.

## How to Run the Code
1.  **Ensure Requirements are Installed:**
    You will need Python installed along with the `pandas` and `matplotlib` libraries. You can install them via terminal using:
    ```bash
    pip install pandas matplotlib
    ```
2.  **Execute the Script:**
    Place the `main.py` and `student_performance.csv` files in the same directory. Run the script using:
    ```bash
    python main.py
    ```
3.  **View Results:**
    The script will display the histogram in a new window and automatically save a copy as `histogram_output.png` in your folder.
